运行步骤:

1.安装MySQL数据库, 执行:

    mysql --user=root -p < schema.sql

2.安装Python3.6, 执行:

    pip3 install -r requirements.txt

    python3 app.py --password=your_database_password

    浏览器访问 http://127.0.0.1:8080

3.在线体验 <http://m.ihouser.com/>
